# 1. hhplus-e-commerce

- java 21
- SpringBoot 3.3.1
- MySql/JPA
- Redis
- Kafaka
- JUnit + AssertJ
- Layered Architecture Based




# 2. 개발 환경 셋팅
redis, mysql, kafka를 전부 `application-setting-docker-compose.yml` (dockerCompose)로 관리하는중

```
docker-compose -f application-setting-docker-compose.yml up -d
```
